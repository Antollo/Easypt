# `getName`

#### `..Root.Object.getName`

Returns name of variable.

* **Return value:** new `String`

## Child of:

[`Object`](docs..Root.Object.md)

## Signatures:

`NativeCallable`, `Callable`

## Example:

```c
var something;
basicOut(something.getName());
basicOut(Object().getName());
```

#### Possible output:

```
something
Anonymous1816
```
