# `toBoolean`

#### `..Root.String.toBoolean`

Checks if string is not empty (true if not empty, false otherwise).

* **Return value:** new `Boolean`

## Child of:

[`String`](docs..Root.String.md)

## Signatures:

`NativeCallable`, `Callable`


## Example:


```c
log("Hello World!".toBoolean());
log("".toBoolean());
```
